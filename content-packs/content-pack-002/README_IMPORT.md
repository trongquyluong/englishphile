# Englishphile Content Pack 002

This pack contains original practice content for Englishphile.

## Totals

- Problems: 55
- Questions: 275
- Split import files: 7

## Recommended import

Use the split files `01` to `07` or run:

```bash
npm run import:pack -- content-packs/content-pack-002
```

Do **not** import `00-all-in-one-content-pack-002.json` together with the split files, because that will create duplicate warnings.

## Suggested workflow

1. Copy this folder into `content-packs/content-pack-002` inside the Englishphile project.
2. Run the CLI import command or use `/admin/import` multi-file upload.
3. Imported content should default to `NEEDS_REVIEW`.
4. Run Content QA.
5. Preview representative problems.
6. Bulk publish only problems without QA errors.
